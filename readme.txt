1. Open the folder where you have installed Wampserver64 on your computer.
(Normally it is located in C:\wamp64)

2. Go to www folder. (C:\wamp64\www)

3. Create a folder called SSD_GoogleDriveUpload.

4. Copy the files in the downloaded SSD_GoogleDriveUpload folder to folder you have created.

5. Run Wampserver64. 

6. Open the browser and type http://localhost/SSD_GoogleDriveUpload/index.html and run the application.
